# qqyjx.github.io

Personal academic homepage of **Yifan Xiao (肖逸凡)** — Ph.D. student in Computer Software at Peking University, working on AI Alignment and large language models.

🔗 **Live:** https://qqyjx.github.io

Bilingual (中文 / English) single-page site. `index.html` is self-contained: inline CSS and a base64-embedded avatar, with web fonts loaded from the Google Fonts CDN. `.nojekyll` disables Jekyll so the page is served as-is.
